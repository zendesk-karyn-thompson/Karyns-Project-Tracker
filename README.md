# Project Portfolio Tracker

A simple, single-page web app for managing and communicating progress across multiple marketing projects.

## Features

- **Portfolio Dashboard**: High-level view of all projects with executive summary
- **Project Detail View**: Comprehensive view with collapsible sections
- **Historical Tracking**: Dated updates that preserve project evolution
- **Value & Impact Summary**: Performance review-ready language generation
- **Local Storage**: All data stored in browser (no backend required)
- **Export/Import**: Backup and restore your data

## Usage

1. Open `index.html` in any web browser
2. Click "Add New Project" to create your first project
3. Fill in details (paste messy notes - they'll be stored as-is)
4. Click into projects to add updates and track history
5. Use "Export Data" to backup your projects

## What This App Tracks

- Project status and priority
- Executive summaries and manager snapshots
- Communication and alignment notes
- Impact and business outcomes
- Risks and blockers
- OKR alignment
- Skills demonstrated
- Historical updates with dates

## Technical Details

- Single HTML file with embedded CSS and JavaScript
- No dependencies or build process
- Data stored in browser localStorage
- Works completely offline
- Mobile responsive

## Customization

To edit labels, sections, or skills:
- Open `index.html` in a text editor
- Section titles: Lines 300-400
- Skills checkboxes: Lines 600-650
- Colors and styling: Lines 20-200

## Performance Review Ready

The "Value & Impact Summary" section automatically translates your work into:
- Business impact language
- Strategic contribution
- Cross-functional influence
- Skills demonstrated
- Measurable outcomes

Perfect for performance conversations and review prep.
